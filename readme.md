## Списки
Нумерованные списки задаются так (уровни можно задать отступами):

1. первый элемент
2. второй элемент
3. третий элемент
4. четвертый элемент
1. пятый элемент
первый элемент
второй элемент
третий элемент
четвертый элемент
пятый элемент
Маркированные списки задаются с помощью дефисов, астерисков или плюсов (уровни можно задать отступами):

+ первый элемент
- второй элемент
+ третий элемент
  - четвертый элемент
  * пятый элемент
первый элемент
второй элемент
третий элемент
четвертый элемент
пятый элемент
